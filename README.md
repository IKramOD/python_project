# New York Airbnb EDA Project

## Project Overview
This project is an Exploratory Data Analysis (EDA) of Airbnb listings in New York City.  
The goal is to understand pricing patterns, neighborhood trends, availability, and factors that influence rental prices.

The project also includes **data cleaning, preprocessing, and visualization** to extract meaningful insights from raw Airbnb data.

---

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
